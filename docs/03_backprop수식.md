# Backpropagation 수식 (길 B)

> STEP 5(길 A, 직관)의 수학 버전. "왜 그 코드가 맞는지"를 이해하기 위한 문서.
> 핵심: 미분 공식은 한 번 이해하고 코드로 박아두는 것. 매번 손으로 풀 일은 없다.
> 디펜스에서 요구하는 건 "개념 설명"이지 칠판 유도가 아니다.

---

## 1. 미분 = 기울기 = dL/dw

> 미분 = "입력을 아주 조금 바꾸면 출력이 몇 배로 바뀌나"의 비율.

`dL/dw` = "w를 조금 바꾸면 loss L이 얼마나 바뀌나" = STEP 4에서 말한 그 기울기.
(분수가 아니라 "w에 대한 L의 기울기"를 뜻하는 기호.)

---

## 2. 알아야 할 기본 미분 3개

```
① 덧셈   z = a + b     →  dz/da = 1
② 곱셈   z = a · w     →  dz/dw = a,   dz/da = w
                          (한쪽을 조금 늘리면 상대방 값만큼 늘어남)
③ sigmoid  a = σ(z)    →  da/dz = a · (1 - a)   ← 자기 출력 a로 표현!
                          (그래서 forward 때 구한 a를 재활용)
```

---

## 3. 연쇄법칙 (chain rule) — backprop의 심장

사슬:  `w → z → a → L`

w를 바꾸면 맨 끝 L이 얼마나 바뀌나?
```
dL/dw = (dL/da) × (da/dz) × (dz/dw)
        └ 끝에서부터 ──────────→ 앞까지 곱한다
```

직관(톱니바퀴): w 1바퀴 → z 3배 → a 2배 → L 5배 돌면, w→L은 3×2×5=30배.
**영향이 곱해지며 전달.** = 길 A의 "영향을 곱하며 뒤로 전파"의 정확한 정체.
곱셈이 끝(L)부터 시작하므로 → 출력→입력 방향(backward).

### 검증된 예시 (뉴런 1개, 제곱오차 loss)
사슬: `x → z=x·w+b → a=σ(z) → L=(a-y)²`,  x=2.0, w=0.5, b=0, y=1.0

```
forward:  z=1.0,  a=σ(1.0)=0.7311,  L=(0.7311-1)²=0.0723

backward:
  ① dL/da = 2(a-y)   = 2(0.7311-1)     = -0.538
  ② da/dz = a(1-a)   = 0.7311×0.2689   = 0.1966
  ③ dz/dw = x        = 2.0
  dL/dw = ①×②×③ = -0.538×0.1966×2.0 = -0.2115
  dL/db = ①×②×1 (dz/db=1)            = -0.1058
```
수치미분(w를 eps만큼 흔들어 L변화 측정)으로 검증 → 정확히 일치.
이 "gradient checking"은 실제 구현에서 backprop 디버깅 표준 기법.

## 4. ★ 출력층의 마법: softmax + cross-entropy

우리 모델 출력층은 softmax + cross-entropy 조합. 이 둘을 함께 미분하면
중간 항들이 전부 약분되어 거짓말처럼 단순해진다:

```
출력층 오차 = 예측 - 정답
```

예시:
```
예측(softmax): [0.27, 0.73]
정답(one-hot): [0,    1   ]   (악성 M)
출력층 오차 = [0.27-0, 0.73-1] = [0.27, -0.27]   ← backprop의 출발점
```

→ 복잡한 미분이 다 사라지고 "빼기 한 줄". 결과만 외워두면 구현에 지장 없음.
(softmax와 cross-entropy가 수학적 짝꿍이라 함께 미분하면 상쇄됨. 그래서 모든
 프레임워크가 이 조합을 기본으로 씀.)

## 5. 전체 backprop을 "코드 모양"으로

```
# === forward (예측) ===
각 층:  z = W @ 입력 + b ;  a = 활성화(z)
출력층까지 흘려 예측 얻음

# === backward (기울기) ===
오차 = 예측 - 정답                       # 1) 출력층 오차 (파트2의 마법)

각 층마다 (출력층부터 거꾸로):
    dW = 오차 ⊗ (이 층 입력)             # 곱셈 미분 dz/dw = 입력
    db = 오차
    오차 = (W.T @ 오차) × a_앞층 × (1 - a_앞층)   # 앞 층으로 전파 (연쇄법칙 + sigmoid미분)

# === update (gradient descent, STEP 4) ===
W = W - 학습률 × dW
b = b - 학습률 × db
```

각 조각의 출처:
- `오차 × 입력`         = 곱셈 미분 (dz/dw = 입력)
- `× a(1-a)`           = sigmoid 미분 (da/dz)
- `W.T @ 오차`          = 오차를 앞 층으로 거꾸로 전달 (연쇄법칙)

---

## 6. 한 줄 요약

```
출력층 오차 = 예측 - 정답
→ 출력에서 입력으로 거꾸로 전파하며 (W.T@오차 × sigmoid미분)
→ 각 층 기울기 dW = 오차 × 입력 을 구하고
→ W = W - 학습률 × dW 로 업데이트
```
미분은 한 번 이해하고 코드로 박아두는 것. backprop 코드는 결국 빼기·곱하기·행렬곱 몇 줄.
